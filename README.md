# Datastructures
Data structures and Algorithms

Initial array :
  
    int nodes[] = { 1, 2, 4, -1, -1, 5, -1, -1, 3, -1, 6, -1, -1 };
  


Pre-Order Traversal :- 
Time Complexity: O( N )

    Output : 1 2 4 -1 -1 5 -1 -1 3 -1 6 -1 -1



In-Order Traversal :- 
Time Complexity: O( N )

    Output : -1 4 -1 2 -1 5 -1 1 -1 3 -1 6 -1


Post- Order Traversal :- 
Time Complexity: O( N )

    output : -1 -1 4 -1 -1 5 2 -1 -1 -1 6 3 1
